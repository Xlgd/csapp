等式3.1: &D[i][j] = xD + L(C * i + j)

所以在C代码中，第3-5行：
int *Aptr = &A[i][0];
int *Bptr = &B[0][k];
int *Bend = &B[N][k];

矩阵A大小为16 * 16， 元素大小为4字节，所以为了得到矩阵A第i行首元素的地址，
使用等式可得：&A[i][0] = xA + 4 (16 * i + 0) = xA + 64i，可以对应汇编代码相应部分

矩阵B大小和元素大小和矩阵A一样，所以为了得到矩阵B第k列的首元素地址，
使用等式可得：&B[0][k] = xB + 4 (16 * 0 + k) = xB + 4k，可以对应汇编代码相应部分

矩阵Bend为矩阵B第k列最后一个元素的地址，使用等式可得：
&B[N][k] = xB + 4 (16 * 16 + k) = xB + 4k + 1024，可以对应汇编代码相应部分
